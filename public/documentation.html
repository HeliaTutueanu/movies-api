API Endpoints

The Movies API stores a vast number of movies and the actors who play in them. You can use it to identify actors/actresses and the movies they have played in.

Request URL Return
GET /movies List of all movies
GET /movies/{title} Movie details (description, genre, director, image URL, featured)
GET /genres/{name} Genre details (description)
GET /directors/{name} Director details (bio, birth year, death year)
POST /users/register JSON: { "username": "example", "password": "password" } User registration status
GET /users/{userID} JSON: { "username": "username" } View user details
PUT /users/update/{userID} JSON: { "username": "newUsername" } User update status
DELETE /users/delete/{userID} User deregistration status
POST /users/{userID}/favorites/add/{movieID} Movie added to favorites
POST /users/{userID}/favorites/remove/{movieID} Movie removed from favorites
